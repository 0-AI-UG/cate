Treemap heatmap. Tile area ∝ total churn, color ∝ change intensity. The brightest, biggest tiles are where the agent did the most work. Live pulse = touched in the last few seconds. Click any tile for the line-level diff.
Activity river. Each lane is a file; each dot is one edit, placed on a shared time axis and sized by the lines changed. You can replay the agent's session and watch edits ripple file-to-file. The playhead scrubs through time.
FileHeat± lines
Tree + heat bars. The familiar git file tree, augmented: a churn bar and live ± counts per file, folders aggregate their children. Lowest learning curve — it slots into the existing explorer. Click a row to expand the diff inline or in the drawer.
Bubble clusters. Files group into directory neighborhoods; bubble size ∝ churn, color ∝ status. Great for spotting where in the architecture an agent concentrated — a whole subsystem lighting up reads at a glance. Orange ring = actively being edited now.
Diff minimap deck. Every touched file is a card with a code-minimap spine — green/red bands show exactly where in the file the edits land, like VS Code's scrollbar overview. Skimmable and information-dense. Click to flip into the full diff.
